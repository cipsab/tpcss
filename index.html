Nuestros Proyectos de Robots Arduino

Robot Seguidor de Línea

Un robot inteligente diseñado para **seguir una línea marcada en el suelo**. Ideal para aprender sobre **sensores infrarrojos** y **control de motores**.

Descripción: Este robot utiliza sensores para detectar la línea y ajustar su movimiento para permanecer sobre ella. Es un excelente proyecto para iniciarse en la robótica móvil.

Robot de Lucha Sumo

Un robot robusto construido para competir en un ring de sumo, intentando **empujar a su oponente fuera del área**.

Descripción: Equipado con sensores para detectar al oponente y potentes motores para el empuje, este robot pone a prueba estrategias de diseño y programación.
